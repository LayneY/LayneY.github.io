Computer Science I


My ISP


Introduction

Our ISP was to create a game using Igis. Our game is called Stick Jump and is a platformer type game.


Concept

A platformer game is a 2D game where the character has to move around and jump to different
platforms on the level.


Motivation

When we were deciding on the kind of game to make, we all recalled playing platform games
such as Mario. This was the motivation for us making a platformer game.


Code

The calculate() function is what deals with gravity in the game. When on a platform, the
character will not be able to go through it unless it is lava.
The moveForward(), moveBackward(), and jump() functions are called when the appropriate
keys are pressed. The forward and backwards functions update each frame of the character
so the character is in the right place. The jump function changes the character's
velocity so that it will go upwards. The calculate function then deals with bringing
the character back down.


Conclusion

We encountered many issues while making this game. The hardest part was making gravity
work as it should, along with trying to make the game fit on a variety of screen sizes.
Overall, however, I am happy with how the game turned out in the end and how our
presentation went.
