My Programming Projects Throughout High School

2023-2024: Eagle Engagement ISP

A new Eagle Engagement application, to replace the broken, outdated one we've been using.

2023: FRC Scouting Application

I helped update the FRC 5417 Scouting Application with new features and update to the new FRC season. The app is designed to simplify scouting matches at FIRST Robotics Competitions. The app makes it easy to see data collected without the use of an Internet connection.

2022-2023: Model Scheduler ISP

An online application used by thousands of Allen High School students to plan their schedules prior to Arena Scheduling opening for the 2023-2024 school year. I learned a lot about JavaScript and frontend design from this project.

2021-2022: Platformer Game ISP

A platformer game run on Coder Merlin IGIS that was my end of year project for Computer Science I. The game is made entirely in Swift and taught me a lot about different programming techniques, especially when trying to implement physics like gravity into the game.
